1. Estruturas de controle

Normalmente: execução em sequência, mas podemos controlar qual será o próximo comando a ser executado if.

1.1. if

O teste deve resultar em um valor true ou false!

Atenção, em C, qualquer valor diferente de zero é verdadeiro (true). 

if( nota >= 6 ) {
  puts("passou");
}

1.2. puts

O comando puts imprime uma string e salta uma linha em seguida.

puts("hello"); é igual à printf("hello");

Use printf quando for imprimir valores em Strings. Use puts quando for imprimir uma String e saltar linha.

1.3. if…else

Podemos usar if/else para especificar duas ações, uma para quando a condição for true e outra para quando a condição for false. 

#include <stdio.h>

int main(void) {
  int nota;

  printf("Digite uma nota [0-10]: ");
  scanf("%d", &nota);

  if( nota >= 6 ) {
    puts("Aprovado");
  } else {
    puts("Reprovado");
  }
  return 0;
}
