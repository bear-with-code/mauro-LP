1. Exibindo dados na web

Aqui temos uma página web básica escrita em linguagem HTML: 

<html>
  <head>
    <meta charset="UTF-8">
    <title>Primeira página</title>
  </head>
  <body>
    <h1>Olá mundo!</h1>
  </body>
</html>

1.1. Testando no servidor

  1. Execute o servidor.
  2. Coloque o código da página no arquivo exemplo.html.
  3. Salve o arquivo na pasta raiz do servidor, geralmente htdocs ou www.
  4. Acesse do navegador: http://localhost/exemplo.html

2. Exibindo usando C

Para criar a página anterior usando C basta imprimir cada linha usando printf(): 

#include <stdio.h>

int main(void) {
    printf("<html>");
    printf("<head>");
    printf("<meta charset=\"UTF-8\">");
    printf("<title>Primeira página</title>");
    printf("</head>");
    printf("<body>");
    printf("<h1>Olá mundo!</h1>");
    printf("</body>");
    printf("</html>");
    return 0;
}

A diferença é que para gerar no servidor adicionamos a linha: 

printf( "Content-type: text/html\n\n" );

Para informar o tipo do conteúdo que está sendo gerado. Os dois \n são obrigatórios: 

#include <stdio.h>

int main(void) {
  printf( "Content-type: text/html\n\n" );
  printf("<html>");
  printf("<head>");
  printf("<meta charset=\"UTF-8\">");
  printf("<title>Primeira página</title>");
  printf("</head>");
  printf("<body>");
  printf("<h1>Olá mundo!</h1>");
  printf("</body>");
  printf("</html>");

  return 0;
}

2.1. Configure o servidor Apache

  1. Abra o arquivo httpd.conf.
  2. Localizar o texto: ScriptAlias cgi-bin "C:/xampp/cgi-bin/"
  3. Inclua o texto abaixo na linha a seguir:
<Directory "C:/xampp/cgi-bin/">
    AllowOverride None
    Options +ExecCGI
    Require all granted
</Directory>

O trecho vai ficar assim: 

ScriptAlias /cgi-bin/ "C:/xampp/cgi-bin/"

<Directory "C:/xampp/cgi-bin/">
    AllowOverride None
    Options +ExecCGI
    Require all granted
</Directory>

</IfModule>

Pare e execute o servidor se já estava funcionando.

2.2. Testando no servidor

  1. Execute o servidor.
  2. Coloque o código anterior no arquivo exemplo.c
  3. Compile gcc exemplo.c -o exemplo
  4. Salve o arquivo na pasta CGI do servidor.
  5. Acesse do navegador: http://localhost/cgi-bin/exemplo

3. Envio de dados

As páginas HTML podem enviar dados usando <form>: 

<!DOCTYPE html>
<html lang="pt-BR">
<head>
    <meta charset="UTF-8">
    <title>Dados</title>
</head>
<body>
    <form action="recebenome">
        <label>Nome:</label>
        <input type="text" name="nome"><br>
        <input type="submit">
    </form>
</body>
</html>

Os dados aparecem na linha de URL no navegador e vão para a página indicada em action. O valor digitado aparece após o nome no formato: nome=fulano ou seja, nome=valor. Assim: 

recebenome?nome=Fulano

3.1. Pastas

As páginas HTML ficam na pasta htdocs e os programas cgi na pasta cgi-bin no caso do XAMPP, assim para enviar dados a página HTML deve usar no action o nome da pasta: 

<form action="cgi-bin/recebenome">

Completa fica assim: 

<!DOCTYPE html>
<html lang="pt-BR">
<head>
    <meta charset="UTF-8">
    <title>Dados</title>
</head>
<body>
    <form action="cgi-bin/recebenome">
        <label>Nome:</label>
        <input type="text" name="nome"><br>
        <input type="submit">
    </form>
</body>
</html>

4. Recebendo dados

Os dados enviados pelo navegador chegam na variável de ambiente QUERY_STRING o tamanho do corpo da mensagem em CONTENT_LENGTH e o tipo de envio em REQUEST_METHOD. 

#include <stdio.h>
#include <stdlib.h>

int main() {
  char *recebido = getenv("QUERY_STRING");
  printf("Recebido: %s\n", recebido);

  return 0;
}

Para exibir qualquer mensagem no navegador precisamos da linha: 

printf( "Content-type: text/html\n\n" );

Resultado: 

#include <stdio.h>
#include <stdlib.h>

int main() {
  printf( "Content-type: text/html\n\n" );

  char *recebido = getenv("QUERY_STRING");

  printf("Recebido: %s\n", recebido);

  return 0;
}

5. Usando os dados

Se o campo for definido assim: 

<input type="text" name="numero">

Então, teremos uma QUERY_STRING assim: 

"numero=12"

Para usarmos o valor 12 precisamos:

  1. Localizar o char '='.
  2. Criar uma string com o conteúdo depois de =.
  3. Transformar a string em inteiro.

5.1. Localizar char

A função strchr(texto, letra) da biblioteca <string.h> localiza um char letra dentro de uma string texto. 

#include <stdio.h>
#include <string.h>

int main(void) {
  char *posicao = NULL;

  char texto[] = "numero=12";
  char letra = '=';

  posicao = strchr(texto, letra);

  printf("Final: %s", posicao);

  return 0;
}

O valor está em posicao+1: 

#include <stdio.h>
#include <string.h>

int main(void) {
  char *posicao = NULL;

  char texto[] = "numero=12";
  char letra = '=';

  posicao = strchr(texto, letra);

  char *valor_recebido = posicao+1;

  printf("Valor recebido: %s", valor_recebido);

  return 0;
}

Podemos transformar a string valor em um int usando atoi() da biblioteca stdlib.h: 

int recebido = atoi(valor);

Ficando assim: 

#include <stdio.h>
#include <string.h>
#include <stdlib.h>

int main(void) {
  char *posicao = NULL;

  char texto[] = "numero=12";
  char letra = '=';

  posicao = strchr(texto, letra);

  char *valor_recebido = posicao+1;

  int numero = atoi(valor_recebido);

  printf("Número: %d", numero);

  return 0;
}
